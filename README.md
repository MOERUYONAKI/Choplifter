# Choplifter

### I - Requirements

**Python 3** or later
> "**Pygame 2.5.0**" lib  
  
### II - Commands for CHPrompt

**1 - /clear**  
> Clear the CHPrompt  
  
**2 - /exit**  
> Close the CHPrompt  
  
**3 - /help**  
> Give the commands list  
  
**4 - /play** *[ -f ]*  
> Start the Choplifter game  
> [ -f ] - Fullscreen mode  
> [ -s ] - Survival mode  
  
**5 - /repo**  
> Return the repository link
