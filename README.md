# Choplifter

### I - Requirements

Python 3 or later
> "Time" lib

### II - Commands for CHPrompt

**1 - /clear**  
> Clear the CHPrompt  
  
**2 - /exit**  
> Close the CHPrompt  
  
**3 - /help**  
> Give the commands list  
  
**4 - /play** *[ -f ]*
